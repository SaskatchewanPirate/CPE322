### Instructions

3. Open a terminal and run the commands specified below
#### Code modified for Windows Command Prompt:
```sh
$ hostname
$ env
$ ps
$ pwd
$ git clone https://github.com/kevinwlu/iot.git
$ cd ~\iot
$ ls
$ cd
$ df
$ mkdir demo
$ cd demo
$ nano file
$ cat file
$ cp file file1
$ mv file file2
$ rm file2
$ clear
$ ver
$ ipconfig
$ ping localhost
$ netstat
```
